
Shabir Ahmad Bhat Blog - Ready to Deploy

1. Replace Google Analytics ID in index.html with your GA ID.
2. Replace all "https://yourbloglink.com" URLs with the actual blog URL.
3. Upload the folder to Netlify:
   - Go to https://app.netlify.com/
   - Sign in with GitHub or email
   - Click "New Site from Git" or "Deploy manually"
   - Drag and drop this folder
4. Set your free domain to shabirahmadbhat.netlify.app
5. Done! Your blog is live.
